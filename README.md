# Data Cleaning with Pandas

This repository contains my practice and learning work on Data Cleaning using Python and Pandas.  
I explored different techniques used to clean and preprocess raw datasets before analysis.

## Topics Covered

- Handling Missing Values
- Handling Duplicates
- Handling Outliers
- Encoding
- One-Hot Encoding
- Changing & Replacing Data Types

Each folder contains examples and practice files related to the specific topic with different datasets and cleaning techniques.

## Tools Used

- Python
- Pandas
- NumPy
- Jupyter Notebook

## Purpose

The main purpose of this repository is to improve my data preprocessing and data cleaning skills, which are essential steps in Data Analysis and Machine Learning workflows.
